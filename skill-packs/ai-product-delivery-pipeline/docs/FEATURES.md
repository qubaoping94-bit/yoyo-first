# 功能说明

`ai-product-delivery-pipeline` 是总控型 skill，用于编排 AI 产品工程化交付流程。

## 能力范围

- 产品需求澄清：用户、场景、数据、安全、上线目标。
- 工程计划：里程碑、验收标准、风险和测试策略。
- 产品实现：前端、后端、测试、环境变量。
- React/Next.js 质量：性能、数据请求、渲染、bundle。
- UI/UX 打磨：响应式、可访问性、加载/空/错误状态。
- AI 架构：Agent、Prompt、评测、监控、Foundry 项目。
- Azure AI：OpenAI、AI Search、OCR、Speech、Document Intelligence。
- AI 网关治理：APIM、限流、缓存、内容安全、成本控制。
- Azure 上线：prepare -> validate -> deploy。
