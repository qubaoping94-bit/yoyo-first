# Third-Party Notices

This package does not vendor third-party skill repositories. It coordinates local or separately installed skills such as:

| Skill | Purpose |
|---|---|
| `douyin-skill-installer` | Extract Douyin content, identify mentioned skills, vet, install, and verify them |
| `skill-vetter` | Security review before installing unknown skills |
| `skill-installer` | Install skills from curated lists or GitHub repositories |
| `skill-creator` | Initialize and validate new Codex skills |
| `writing-skills` | Guidance for writing reusable skill instructions |
| `github-skill-pack-publisher` | Package, zip, commit, push, and confirm GitHub publication |
