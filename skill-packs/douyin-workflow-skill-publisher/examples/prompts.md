# Prompt Examples

## Full Workflow

```text
用 $douyin-workflow-skill-publisher 处理这个抖音链接：安装里面提到的 skill，判断能不能组成工作流；如果可以，生成组合 workflow skill，并上传到我的 GitHub 仓库。
```

## Install Only, Then Decide

```text
用 $douyin-workflow-skill-publisher 看这个抖音视频里提到哪些 Codex skill。先安装和校验，再告诉我这些 skill 是否适合组成一套工作流。
```

## Publish Rule Reminder

```text
用 $douyin-workflow-skill-publisher 把这次抖音 skill 工作流保存成 skill。上传时按默认规则：每个 skill 单独一个技能包，不要合并。
```

## When Metadata Fails

```text
这个抖音链接可能打不开字幕。用 $douyin-workflow-skill-publisher 通过页面标题、画面抽帧和可见文字识别视频里出现的 skill，再继续安装和打包。
```
