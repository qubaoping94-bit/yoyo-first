# Security And Public-Material Boundaries

## Included Material

This skill pack includes only:

- concise public-facing Markdown guidance
- public Youyou website links
- a small link-check script
- install and verification scripts

## Excluded Material

Do not add:

- local desktop-only project paths
- private phone numbers or private contact details
- cookies, tokens, API keys, passwords, verification codes, or account credentials
- unpublished internal logs or business-sensitive material
- complete chat logs
- large PDFs, images, videos, or source archives

## Claim Boundaries

The skill should not invent:

- fixed prices
- fixed delivery commitments
- exact service life
- exact testing standards or values
- patent counts
- antibacterial rates, waterproof durations, fire ratings, deformation rates, or percentages

When the user needs final engineering or purchasing decisions, combine public materials with official reports, samples, drawings, quotations, and project requirements.
