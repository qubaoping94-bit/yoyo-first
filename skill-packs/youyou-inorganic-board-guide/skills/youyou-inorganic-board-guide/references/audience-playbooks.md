# Audience Playbooks

Use this file to adapt the product-consultant experience to the user's role.

## 1. First-Time Customer

Goal: build trust and basic understanding quickly.

Start with:

- what Youyou is
- why it is a system rather than one board
- where to look first

Recommended answer pattern:

```text
优优无机板可以先按“一个核心材料，四类应用”理解：无机板、无机门、无机墙板、无机复合家具板，最后组合成门墙柜一体化无机空间。第一次了解建议先看总入口，再看产品演化页。
```

Best links:

1. https://youyou-sites-list.pages.dev/
2. https://youyou-ai-gallery.pages.dev/product-evolution/

## 2. Dealer Or Channel Partner

Goal: help them forward materials without overexplaining.

Ask or infer:

- Is the customer new, comparing materials, or asking one specific concern?
- Is the dealer sending a first message, follow-up proof, or朋友圈 content?

Recommended route:

1. First message: website hub.
2. Visual follow-up: AI gallery.
3. Specific concern: one product-evolution page.
4.朋友圈: moments copy site.

Tone:

- concise
- confident
- not technical-heavy
- ready to paste into WeChat

## 3. Project Or Selection Scenario

Goal: match the product system to the use case without pretending to be a final engineering spec.

Ask one question if missing:

```text
这个项目主要用在门、墙、柜，还是整装空间？不同部位推荐看的资料会不一样。
```

Then route:

- door/wall/cabinet system: product-evolution hub
- damp or bathroom/kitchen: waterproof and mildew-proof pages
- hotel/office quietness: sound-insulation page
- long-term stability: composite lifetime/performance page

## 4. Showroom Or Presentation Use

Goal: help the user explain the product live.

Recommended structure:

1. Opening sentence: `我们不是只看一张板，而是看它如何变成门、墙、柜和一个完整空间。`
2. Show material origin or core board page.
3. Show one concern-specific demo page.
4. End with website hub or gallery for follow-up.

## 5. Content Operator

Goal: create shareable content without making unsupported claims.

Use:

- moments copy site for variants
- gallery for visuals
- response templates for copy structure

Avoid:

- fake test numbers
- invented patents
- exaggerated "industry first" statements
- private contact details
