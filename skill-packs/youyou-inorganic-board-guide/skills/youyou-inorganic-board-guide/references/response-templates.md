# Response Templates

Use these templates to make answers polished and consistent. Adapt wording to the user's role and scenario.

## Link Recommendation Card

```text
建议先看：<页面名称>
链接：<URL>
为什么看它：<一句话说明它解决的问题>
下一步：看完后告诉我你更关心 <场景/性能/价格/施工/经销商转发>，我再帮你选下一页。
```


## Comparison With Ordinary Panels

```text
更稳妥的讲法是：普通板材往往是单一材料或单一部位选择，优优无机板强调从材料、结构到门墙柜一体化空间的系统能力。它的价值不是一句“比谁都好”，而是让客户看到材料逻辑、产品结构和应用场景。
```

## Scenario Selection Question

```text
为了给你推荐最合适的资料，我先问一个关键问题：你主要是给客户看整体产品体系，还是客户已经在问某个具体问题，比如防潮、防霉、阻燃、开裂变形、价格或施工？
```

## High-End Short Pitch

```text
优优无机板的核心价值，是把无机材料从“一张板”升级为门、墙、柜和整装空间系统。客户看到的不只是一个材料卖点，而是一套能用于长期居住、商业空间和项目交付的产品逻辑。
```
