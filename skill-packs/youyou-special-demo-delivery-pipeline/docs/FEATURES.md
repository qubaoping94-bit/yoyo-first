# Features

## End-to-End Topic Delivery

The skill covers the full Youyou special-topic page workflow:

- interpret a Chinese topic title and multiple reference posters
- create a stable slug, short title, use case, tags, and central thesis
- generate or revise a master presentation image
- build a local HTML demo page with fullscreen and zoom support
- register the page in the product-evolution hub
- update the site directory and WeChat share copy/assets
- prepare the Cloudflare public folder
- deploy to the gallery, backup gallery, and site-list projects
- run local and public audits before final handoff
- update Obsidian Codex memory when the result is reusable

## Claim Boundaries

The skill explicitly avoids unsupported metrics, standards, patent counts, service-life claims, percentages, and test values unless the user supplies reliable evidence.

## Preservation Rule

When revising an existing image, preserve unrelated illustrations and content. Only change the requested section, such as smoke toxicity, antibacterial content, noise isolation, or lifetime-performance messaging.

## Project Fit

This is project-specific to the Youyou inorganic-board `product-evolution` delivery chain under the user's Windows workspace. It is intended for repeated dealer-demo page production, not generic website development.
