# 功能说明

`youyou-demo-page-delivery-pipeline` 是优优无机板演示系统页面的完整交付 skill。

## 核心能力

- 读取用户主题、参考图和改动要求，形成页面 brief。
- 复用小黑长卷、近白材料摄影棚、黑场发布会 HTML 等既有视觉标准。
- 生成或修订最终主图，并记录尺寸、比例和输出路径。
- 创建全屏发布页，支持按钮、双击和 `F` 进入全屏。
- 追加产品演化页配置，接入优优 AI 图库产品演化总站。
- 刷新网站总汇、微信分享图和微信发送话术。
- 构建、部署并运行本地和公网审计。
- 在任务结束前更新 Obsidian 长期记忆。

## 固定边界

- 不编造检测数值、百分比、标准编号、杀菌率、寿命承诺或医疗化结论。
- 不用网页层文字随意覆盖主图错误文字，除非用户明确要求。
- 不跳过移动端和桌面端溢出检查。
- 不把用户无关项目文件或敏感信息打包进发布仓库。
