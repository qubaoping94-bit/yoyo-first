# Prompt Examples

## Build a Pilot

```text
Use $customer-acquisition-pipeline to build a 20-lead pilot for my AI automation consulting service. Target US B2B SaaS founders with small teams. Put the leads into my Feishu Base and draft a first email plus two follow-ups. Do not send anything.
```

## Continue From Lark Base

```text
Use $customer-acquisition-pipeline to continue the leads in my Lark Base. Focus on records with Status = new or needs-review, enrich missing evidence, and draft outreach only for high-fit leads.
```

## Scrape Specific Sources

```text
Use $customer-acquisition-pipeline with these source URLs. Extract companies that match my ICP, dedupe them by domain, create Lark Base records, and generate a short Chinese outreach draft for each.
```

## Review Drafts

```text
Use $customer-acquisition-pipeline to review the drafted emails in my customer table. Mark weak personalization as needs-review and prepare a summary of which drafts are ready for human approval.
```

## Chinese Workflow

```text
用 $customer-acquisition-pipeline 做一条获客流水线：先找公开线索，再写入飞书多维表格客户库，最后生成冷邮件草稿；不要自动发送。
```
